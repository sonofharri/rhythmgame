Custom Map Instructions

  1. Install pyperclip using pip
  2. Get the python script by clicking here
  3. Put it in a new folder
  4. Find some osu mania maps
  5. Change the file extension to .zip
  6. Go inside and take the .osu files
  7. And put them in the same directory as the python script
  8. Then run the python script
  9. Put in the numerical id that it gives you for the map you want
  10. Paste it into a text editor, and replace the "song" parameter at the beginning with the song url
  11. Go back to cmu and paste that into console
